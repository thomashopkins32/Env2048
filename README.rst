======
Env2048
======

OpenAI Gym environment for the game 2048!

* Free software: 3-clause BSD license

Features
--------

* Any size game board!
* `render()` support!
* Different reward types!

Installation
------------

To try it out yourself:

.. code-block:: bash

    $ git clone https://github.com/thomashopkins32/2048AI.git
    $ cd 2048AI
    $ pip install -e .
